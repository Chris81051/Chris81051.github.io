[教學] Hexo 嵌入 Facebook 留言板 外掛

發表於 2022-11-21 分類於 Hexo 閱讀次數：
特色：可通用所有的主題，不需要修改主題檔案，能決定留言板放的位置，使用者比較大眾通用性較好

前言

　　筆者在 Hexo 留言板考慮了很久，一直在評估要用哪款，市面上有很多像是 Valine、Gitalk、Cusdis、Disqus 等等各有優缺點，有些整合了許多社交帳號登入留言，有些不用登入就可直接留言等等，另外也有考慮到留言的託管網站，有部分是內地的網站在託管，有些是很大的品牌在管理，像是 GitHub、FB 的大廠在維護。

　　最後選擇了 FB，相較於其他品質較好外，私心覺得看繁體比較舒服，其實 FB 的留言板也有優點，是使用者比較大眾通用性較好，另外他有後台可以管理留言，有相當好的整合。

使用套件

依照自己使用的套件管理工具選擇指令

配置 Hexo 設定檔

在專案的根目錄 _config.yml

配置留言板位置

在你的文章裡添加留言板標籤。(適用在 MD 檔)

如果嫌每次都要加標籤太麻煩的話，可以配置在預設模板 scaffolds 中，以後產生指定 layout 的 MD 時就會預設附在裡面。

配置 Facebook 端

Hexo 方面都設定完後，我們來配置 Facebook 端的設定
需先用的 Facebook 帳號開通 FB 開發者模式
https://developers.facebook.com/products/social-plugins/comments?locale=zh_TW

開通 FB 開發者模式後，最主要是取得 ID 號碼，我個人有把應用程式創建完，不用創建應用程式也會有 ID

把你的應用程式 ID，填在上述 Hexo 的 appId 裡

你想創建應用程式的話這樣做，選擇 Webside (詳細要填的東西是 Facebook 想要蒐集你的資料)，一直案繼續就可以

管理留言

https://developers.facebook.com/tools/comments/
